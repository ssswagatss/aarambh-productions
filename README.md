### Vercel (via GitHub)

This project is optimized for zero-config deployment on Vercel.

1.  Push this repository to GitHub.
2.  Log in to [Vercel](https://vercel.com).
3.  Click **"Add New..."** > **"Project"**.
4.  Import your `aarambh-weddings` repository.
5.  Click **Deploy**.

Vercel will automatically detect the static project and serve `index.html` as the root.
